# nezu-app (IPA ビルダー)

[English](./en/README.md)

GitHub Actions を使用して署名無し IPA をビルドし、iOS 26 Liquid Glass デザインを採用した次世代モバイルアプリを構築するプロジェクトです。

詳細な機能、ビルド手順、および設計仕様については、[`docs/`](./docs/README.md) フォルダ内の各ドキュメントを参照してください。

- **[ビルドプロセス (Build Process)](./docs/build-process.md)** ([English](./docs/en/build-process.md))
- **[更新機構 (Update Mechanism)](./docs/update-mechanism.md)** ([English](./docs/en/update-mechanism.md))
- **[デザインガイド (Design Guide)](./docs/design-guide.md)** ([English](./docs/en/design-guide.md))

## 🚀 クイックスタート

1. `main` ブランチにプッシュすると自動ビルドが開始されます。
2. GitHub の **Releases** タブから最新の IPA をダウンロードできます。
3. SideStore などの署名ツールを使用して iOS 実機にインストールしてください。

*詳細は [docs/README.md](./docs/README.md) を参照してください。*
